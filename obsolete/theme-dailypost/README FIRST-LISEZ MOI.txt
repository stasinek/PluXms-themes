* Installation
**************
Le thème nécessitant l'utilisation de plugins, l'archive contient un dossier pluxml v5.1.6 directement utilisable.

** Première mise en ligne :
**************************

Ces instructions concernent ceux qui n'ont pas de site et qui souhaitent le mettre en ligne pour la première fois.

Placer le dossier pluxml à la racine de votre site. N'UTILISEZ PAS LES FICHIERS A LA RACINE DE L'ARCHIVE QUI SONT VIDES.

Connectez-vous à l'adresse du site et suivez les instructions d'installation.

Une fois l'installation terminée, la page d'accueil s'affiche avec le thème par défaut.
Connectez-vous au panneau d'administration et cliquez sur Paramètres.

Dans les options d'affichage, sélectionnez le thème "dailypost".

Puis rendez-vous dans la rubrique plugins, sélectionnez les et activez les tous.

** Mise à jour d'un pluxml déjà en ligne :
******************************************
Ces instructions concernent ceux qui ont déjà un site et qui souhaitent le mettre à jour et utiliser ce thème.

Dans le dossier pluxml, ouvrez le dossier themes/ et copier le dossier dailypost/ dans le dossier themes/ de votre pluxml en ligne.
Copiez l'ensemble des dossiers présents dans le dossier plugins/ dans le dossier plugins/ de votre pluxml en ligne. Si vous ne souhaitez pas tous les utiliser, seul le plugin "about" est absolument nécessaire. Le plugin plxMyAllArchive est également modifié afin que l'affichage de la page allarchive se fasse correctement. Il est donc fortement conseillé d'utiliser la version fournie avec l'archive.


* Rubrique "A PROPOS"
*********************
Le plugin "About" permet de renseigner la rubrique "A PROPOS" située dans la sidebar ainsi que les images de la rubrique "A VOIR" également dans la sidebar.

Il permet également d'avoir des sous-menus dans le menu principal et l'affichage de commentaires imbriqués, ainsi que l'affichage de liens "précédent" "suivant" au pied des articles.


Enjoy ;-)
