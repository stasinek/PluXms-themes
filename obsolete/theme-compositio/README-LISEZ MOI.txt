* Installation
**************
Placer le dossier pluxml à la racine de votre site.

Connectez-vous à l'adresse du site et suivez les instructions d'installation.

Une fois l'installation terminée, la page d'accueil s'affiche avec le thème par défaut.
Connectez-vous au panneau d'administration et cliquez sur Paramètres.

Dans les options d'affichage, sélectionnez le thème "compositio".

Puis rendez-vous dans la rubrique plugins et activez les tous. Le plus important est le plugin about qui permet un affichage imbriqué des commentaires et un menu principal avec des sous-rubriques correspondant aux articles des catégories.


* Le Plugin "about"
*********************
Le plugin "About" permet de renseigner la rubrique "A PROPOS" ,située dans la sidebar par exemple, ainsi que les images et les liens de la rubrique "A VOIR" également dans la sidebar.

La page de configuration permet de configurer les paramètres généraux du plugin.

La page d'administration, accessible par le lien situé dans la sidebar de l'administration de pluxml, permet de définir le texte de la rubrique "A propos" ainsi que les caractéristiques des images de la rubrique "A VOIR".

Le plugin permet également l'affichage des commentaires imbriqués. Compositio est configuré par défaut pour cet affichage.

Si vous souhaitez utiliser ce plugin avec un autre thème, consultez la page d'aide pour avoir un exemple de page "commentaires.php". Il faudra peut-être également modifier le rendu de la méthode "formatComments" dans la page "about.php".




Enjoy ;-)
